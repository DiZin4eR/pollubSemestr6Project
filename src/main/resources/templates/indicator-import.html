Supported data

Indicators

Use files with a top-level subjects array or exported subjectTree data.

Variable data

Use JSON exported from a variable data page. It must contain variableId and regions.

Example JSON

A full import should include indicator subjects and matching variable data. The subject referenced by variableData.subjectId must exist in the same file or already be stored. 

{
  "subjects": [
    {
      "id": "KIMP_FULL",
